Анализ зарплат аналитиков

Дашборд с визуализацией данных о зарплатах аналитиков из hh.ru

О проекте

Собранный дашборд показывает текущую ситуацию на рынке труда для аналитиков. Данные обновляются регулярно.

Что можно увидеть: количество вакансий, долю удалённой работы, средние зарплаты и их распределение по городам на карте.

Как сделано

PostgreSQL Power BI Python ETL

Что внутри

Основные блоки

  • KPI сверху – ключевые цифры по вакансиям и зарплатам
  • Фильтры слева – можно отбирать по формату работы, опыту и роли
  • Карта – города с вакансиями, размер точки зависит от количества

Анализ зарплат

  • Как зарплата зависит от опыта (от junior до senior)
  • Сравнение по ролям: BI, продуктовые, системные аналитики
  • Распределение по городам и форматам работы
